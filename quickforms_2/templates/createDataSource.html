Create a Database Data Source

Paste the following code in tomcat/conf/context.xml

<Resource driverClassName="com.microsoft.sqlserver.jdbc.SQLServerDriver" maxActive="15" maxIdle="5"
maxWait="5000" name="jdbc/[[appName]]" password="examplePassword" type="javax.sql.DataSource"
url="jdbc:sqlserver://localhost:1433;autoReconnect=true" username="exampleUsername"/>

Make sure to change the "name" parameter to the name of the database
Wait for 30 seconds after saving, then navigate to http://localhost/quickforms to validate the connection

Get Result Set (getFactData)

Example REST Request

http://www.servername.com/quickforms/getResultSet?app=example&queryLabel=getCarsByRating&params=
Params should be a comma separated list of filters
Ex. params=car=Audi,userId=4

Example Response

[{"id":1,"form":"test","Car Type":"Mazda","My Rating":"80%"},{"id":2,"form":"test","Car Type":"Volvo","My Rating":"90%"},{"id":3,"form":"test","Car Type":"Audi","My Rating":"40%"}]
